## Common Questions

### "Why isn't it showing code examples?"

This system is designed for business users and automatically filters out technical code. If you need technical details, contact the development team.

### "How current is the information?"

The knowledge base is updated periodically. For the latest IFRS standards or recent feature changes, verify with the most recent documentation.

### "Can I save my questions and answers?"

Currently, the system doesn't save history. Copy important answers to your own notes if needed.

### "What if I don't get a good answer?"

Try:
1. **Rephrasing** your question
2. **Being more specific** about what you need
3. **Breaking down** complex questions into smaller parts
4. **Checking the sources** - they might have the info even if the answer doesn't
