# Consolidation Methods

Documentation for the three primary consolidation methods supported by Prophix.Conso.

## Documents

| Document | Description | IFRS Standard |
|----------|-------------|---------------|
| [global-integration.md](global-integration.md) | Full consolidation for controlled subsidiaries (>50% voting rights) | IFRS 10 |
| [equity-method.md](equity-method.md) | One-line consolidation for associates (20-50% influence) | IAS 28 |
| [proportional-method.md](proportional-method.md) | Proportional consolidation for joint ventures | IAS 31/IFRS 11 |
| [consolidation-method-determination.md](consolidation-method-determination.md) | Threshold-based method selection logic | IFRS 10 |

## Key Concepts

- **Control**: >50% voting rights triggers Global Integration (G)
- **Significant Influence**: 20-50% triggers Equity Method (E)
- **Joint Control**: Shared control triggers Proportional (P) or Equity Method
- **No Consolidation**: <20% or excluded entities (N)
