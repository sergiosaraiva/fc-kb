## Key Concepts

- **Goodwill** = Fair value of consideration - Net identifiable assets acquired
- **Minority Interest (NCI)** = Non-controlling ownership share of subsidiary equity
- **Circular Ownership** = When Company A owns B which owns A (direct or indirect)
