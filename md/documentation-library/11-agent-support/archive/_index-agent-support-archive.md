# Agent Support Archive

## Overview

This folder contains deprecated or superseded agent support files that have been archived as part of quality improvement initiatives.

## Archive Policy

Files are archived (not deleted) when:
- Content has been consolidated into other files
- Functionality has been replaced by improved implementations
- Files are no longer actively used but may contain historical reference value

## Current Contents

| File | Archived Date | Reason | Replaced By |
|------|---------------|--------|-------------|
| `error-handling-patterns.yaml` | 2024-12-04 | Content consolidated | `api-error-catalog.yaml`, `api-debugging-patterns.yaml` |

## Usage Guidelines

**DO NOT** use files from this archive folder for:
- Active agent implementations
- Production knowledge base queries
- New development work

**DO** reference archive files for:
- Historical context on design decisions
- Understanding evolution of knowledge structures
- Migration reference when updating dependent systems
