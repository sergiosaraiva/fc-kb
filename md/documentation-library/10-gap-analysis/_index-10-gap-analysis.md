## Summary

Features documented in Allen White's "Direct Consolidation" that have limited or no support in Prophix.Conso:

| Feature | Status | Workaround |
|---------|--------|------------|
| Step Acquisition | 50% | User Eliminations |
| Treasury Shares | 60% | User Eliminations |
| Preference Shares | 55% | Single class model |
| Impairment Testing | 55% | Manual process |
| Hyperinflation (IAS 29) | 60% | Simplified adjustment |

## Gap Categories

2. **UI Gaps**: Features with no direct user interface
3. **Automation Gaps**: Features requiring manual intervention
