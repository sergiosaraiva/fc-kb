## User Guide

Account analysis reports
On the Analysis Reports Accounts page, you have the ability to generate multiple analysis reports or Pivot Table Reports specifically focused on the consolidated data per account. These reports provide valuable insights and allow for in-depth analysis of the financial information pertaining to individual accounts.

To access the Analysis Reports Accounts page, click Reports > Analysis Reports > Accounts.

Generate an account analysis report:
1. In the Journal Summation field, select a journal view to use for generating the report.
2. Optionally, to include companies that are not consolidated in the report, click Include Not Consolidated Companies.
3. From the list of Reports, select the report you want to generate.
4. To see a sample of the report you selected, click the sample icon for that report. The sample is only a representation and does not reflect the data in your database.
5. Click Generate to generate the selected report.

Generate an analysis report:

2. In the Custom Report field, select the user reports to use as the layout of the analysis report.
3. From the list of Reports, select the report you want to generate.
4. To see a sample of the report you selected, click the sample icon for that report. The sample is only a representation and does not reflect the data in your database.
5. Click Generate to generate the selected report.
