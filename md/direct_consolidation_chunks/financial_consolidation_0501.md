# 6.4 Flows for the consolidated cash flow statement

We know that the main objective of a consolidation is to show through a set of structured accounting reports what a group of companies has done with the outside world. This economical (non fiscal) picture is provided through balance sheet, P&L and a number of notes to the accounts. But now, we want to know what this group did with its cash?

In other words, is it possible to build a cash flow statement for a group of companies in a similar way we proceed for a single company?
