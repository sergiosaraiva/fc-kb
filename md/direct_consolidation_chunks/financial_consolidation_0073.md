# Chunk 73: Consolidation Methods:

## Context

This section covers the equity method for 20-50% ownership stakes.

## Content

## Consolidation Methods:
- **If joint control agreement**: Each party uses proportional consolidation (33.33%)
- **If no joint control**: Each party uses equity method
- **If one party has casting vote**: That party consolidates globally
