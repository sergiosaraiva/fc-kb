# Chunk 134: The percentage of control

## Context

This section covers the equity method for 20-50% ownership stakes. Key thresholds: control (>50%) triggering global integration, significant influence (20-50%) requiring equity method, financial investment (<20%) using cost method. Shows detailed calculations.

## Content

### The percentage of control

The notion of percentage of control has already been widely commented here above. It is recalled that the control percentage determines the consolidation method. Therefore, it must be calculated for each company and for each consolidation.

The following table lists the basic principles that are to be applied to each individual company:


| Control percentage | Level of control | Consolidation method |
| :--- | :--- | :--- |
| >50% | Exclusive control | Global integration |
| 50% with existence of another shareholder with whom control is shared (*) | Joint control | Proportional Integration (or equity method if IFRS) |
| â‰¥20% and <50% | Significant influence | Equity method |
| < 20% | No significant influence | Not consolidated |


(*) May also occur with more than two shareholders. The essence here is that the control is shared between a limited number of shareholders.

Please note that exceptions are always possible, though one must comply with the requirements of the reference framework that was chosen for the

purpose of drawing up the consolidated accounts, whether it's Local Gaap or IFRS.
