# Chunk 68: Consolidation Benefits:

## Context

This section describes global integration method for controlled subsidiaries.

## Content

## Consolidation Benefits:
- Strong control minimizes conflicts
- Limited minority interest issues
- Efficient decision-making structure


It may happen that a parent company has directly and indirectly control at the same time. We then add the percentages.

In the case of sole control, consolidation is, in principle, done via the global integration method.
