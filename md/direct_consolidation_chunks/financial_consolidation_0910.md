## Content

### A spreadsheet bundle or a web bundle?

Thanks to the internet technologies, a web bundle is progressively replacing spreadsheet bundles despite the fact that both solutions are relatively equivalent.

It must also be pointed out that using spreadsheet bundles generally requires more manual preparations that giving a web access to each subsidiary.

When using a spreadsheet bundle, it is easy for Auditors to receive a copy for approval. The same advantage should be maintained when using a web bundle by giving them a "read only" access.
